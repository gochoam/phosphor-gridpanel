Static maxSizeProperty

maxSizeProperty: Property<Spec, number>
  • Defined in index.ts:762

The property descriptor for the maximum size.

The row or column will never be resized more than this size.

Notes

This value is clamped to a lower bound of 0.

The minSize takes precedent when in conflict.

The default value is Infinity.

See also: maxSize

Static minSizeProperty

minSizeProperty: Property<Spec, number>
  • Defined in index.ts:741

The property descriptor for the minimum size.

The row or column will never be resized less than this size.

Notes

This value is clamped to a lower bound of 0.

This takes precedence over maxSize when in conflict.

The default value is 0.

See also: minSize

Static sizeBasisProperty

sizeBasisProperty: Property<Spec, number>
  • Defined in index.ts:721

The property descriptor for the size basis.

This controls the size allocated to the row or column before the stretch factor, size limits, and surplus/deficit space is taken into account.

Notes

The default value is 0.

See also: sizeBasis

Static stretchProperty

stretchProperty: Property<Spec, number>
  • Defined in index.ts:782

The property descriptor for the stretch factor.

This controls how much the row or column is resized relative to its siblings if there is surplus or deficit layout space.

Notes

This value is clamped to an integer with a lower bound of 0.

The default value is 1.

See also: stretch
